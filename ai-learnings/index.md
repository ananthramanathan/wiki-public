### 2026-04-14
Designed a personal knowledge wiki architecture on GitHub. Two repos (wiki-public, wiki-private), flat file structure, /wiki command protocol. Optimised for AI read/write and token efficiency over visual polish. Key insight: a flat .md file pulled at session start costs ~200–500 tokens vs 3–5x that with any API-based tool like Notion. SSH works better than HTTPS for private GitHub repos when using Claude Code CLI.
